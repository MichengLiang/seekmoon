# Code of Conduct

This project uses the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

Participants are expected to make contribution spaces respectful and focused on
the project objects: the SeekMoon CLI, the package-reuse bookshelf, and related
documentation.

## Expected Behavior

- Use clear, relevant, and evidence-based communication.
- Keep criticism focused on code, documentation, contracts, tests, and behavior.
- Respect different levels of familiarity with MoonBit, package ecosystems, Go,
  and AsciiDoc.
- Avoid publishing private information, credentials, or sensitive local paths.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Deliberate disruption of issues, pull requests, reviews, or discussions.
- Publishing private information without permission.
- Introducing malicious code, misleading reports, or knowingly false evidence.

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or restrict
participation when behavior prevents productive project work.
