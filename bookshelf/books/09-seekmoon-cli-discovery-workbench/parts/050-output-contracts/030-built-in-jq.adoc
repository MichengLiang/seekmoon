== Built-in jq

Built-in jq 是 SeekMoon 的输出模式。它把当前命令的 JSON output 发送给本机 `jq` 可执行文件，并通过 subprocess 返回 jq result。

[source,bash]
----
seekmoon search markdown --jq '.results[].module'
seekmoon api 1 --package src/api --jq '.types[].name'
seekmoon source 1 --jq '.selected_source.path'
----

求值顺序固定：

[source,text]
----
command input -> canonical object -> JSON output -> jq subprocess stdin -> jq result
----

`--jq` 不读取 pretty text。除非命令选项改变来源加载规则，`--jq` 不改变 evidence loading；它只转换 JSON projection。

`--jq` 执行前检查当前运行环境中的 `jq`。`jq` 可用时，SeekMoon 把 JSON output 写入 jq subprocess 的 stdin，并把 jq stdout 作为命令输出。`jq` 不可用时，命令进入 error surface；错误对象标记为 `jq`，状态为 `unavailable`。

示例：

[source,bash]
----
seekmoon search markdown --jq '.results[] | select(.license.value == "MIT") | .module'
----
