== WP09：CLI Command Surface

=== 工作包身份

WP09 实现 SeekMoon 的 Cobra command surface。它定义 root command、subcommands、flags、input parsing、output mode options、thin command handlers and process exit mapping。它不实现 source reading、business service composition or rendering internals。

=== 实现边界

Cobra types remain in `internal/cli`. Command handlers validate surface syntax, build input object, call one service method and pass result/error to renderer. They do not construct HTTP URLs、read files、run Moon CLI、format tables or write records directly.

Command parameters use object names: query、module coordinate、package path、candidate number、target、record conclusion。Endpoint URLs, asset URLs, local paths and cache paths remain internal implementation details。

=== 测试责任

WP09 tests cover:

* Command registration and help surface includes all command names。
* Required argument and flag parse failures map to exit code `2`。
* Output mode flags are accepted on output-capable commands。
* `--jq` requires expression。
* Candidate number and module coordinate syntax produce distinct input objects。
* Thin handler calls expected service method once。
* Service error is passed to renderer and maps to exit code `1`。

=== 审核尺子

审核者检查五项。

* `internal/cli` imports Cobra, model/service/output input types, but not source readers。
* Command handlers follow parse args -> build input -> context -> service -> render -> exit mapping。
* Help surface teaches command syntax, not hidden implementation URLs。
* Output mode handling is shared and consistent。
* Process exit and evidence state remain separate objects。
