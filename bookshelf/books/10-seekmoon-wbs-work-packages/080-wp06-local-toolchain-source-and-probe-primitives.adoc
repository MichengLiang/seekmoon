== WP06：Local Toolchain Source and Probe Primitives

=== 工作包身份

=== 实现边界

Moon CLI commands record command、cwd、environment summary、exit status and log path. Command failure does not rewrite upstream facts. `moon add` success is not adoption success. `moon check` and `moon build --target` results are local derived evidence bound to environment、version and target。

Local registry index is JSON Lines. WP06 normalizes only fields represented in current model and preserves raw line for audit. Local cache hit is current machine fact; it does not imply another machine has the same cache.

Project context can enrich search, probe, record and report. Read failure does not block global search, but affects contextual probe and report. Probe mutation defaults to `.seekmoon/probes/<id>`.

=== 测试责任

WP06 tests cover:

* `moon --version` command result mapping。
* Non-zero `moon fetch` and `moon build` produce `failed` command result and log path。
* Local index JSONL parser handles sparse records and malformed lines。
* Project context reader handles TOML and JSON file variants。
* Probe path construction is deterministic and project-bounded。
* Core source reader distinguishes present、unavailable and failed filesystem states。

=== 审核尺子

审核者检查五项。

* Runner calls are context-aware and log-bearing。
* Local command failure is source evidence, not upstream metadata mutation。
* Probe default path lives under `.seekmoon/probes/`。
* Project manifest mutation is not hidden in local primitives。
* Target support without metadata or probe remains `unknown`。
