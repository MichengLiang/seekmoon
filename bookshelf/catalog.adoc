== 08 的知识结构

* xref:books/08-package-reuse-ecosystem/book.adoc#part-orientation[进入包复用生态]：定义研究对象、读者路径和基本命题。
* xref:books/08-package-reuse-ecosystem/book.adoc#part-object-model[对象模型]：定义 Package、Artifact、Manifest、Registry、Index、Resolver、Lockfile、Provenance 和 Attestation。
* xref:books/08-package-reuse-ecosystem/book.adoc#part-lifecycle[生命周期]：描述 Authoring、Declaration、Publication、Indexing、Discovery、Assessment、Resolution、Realization、Maintenance 和反馈回路。
* xref:books/08-package-reuse-ecosystem/book.adoc#part-evidence-measurement[证据与度量]：区分证据源、维度、变量、指标、权重和分数。
* xref:books/08-package-reuse-ecosystem/book.adoc#part-ecosystem-studies[主流生态对照]：对照 JS/TS、Rust、Python、Go、JVM/Kotlin、.NET、C/C++、Swift 和跨生态数据表面。
* xref:books/08-package-reuse-ecosystem/book.adoc#part-security-governance[安全与治理]：拆分漏洞状态、发布者身份、构建来源、制品完整性和注册表治理。
* xref:books/08-package-reuse-ecosystem/book.adoc#part-evaluation-scale[评价尺度]：说明变量组、作用域、归一化、可比性边界和分数读取方式。

== 维护参考

编辑 8 号书时，使用 xref:books/07-structured-writing-conventions/book.adoc[07 结构化书写约定] 检查以下规则：

* 标题层级必须连续。
* 被交叉引用且标题文本不稳定或不唯一的节点需要 stable ID。
* 跨越书名、部、章等大跨度的显式指代需要 xref。
* 关系谓词必须同时进入正文和结构词表。
* 后置附录、术语表、参考文献和索引需要承担查阅功能。
